## ===== Installation =====

## Npm
    $ npm install lunik-torrent
##### Exemple
    require('lunik-torrent')

## Git
    $ git clone https://github.com/Lunik/Lunik-Torrent.git
    $ cd Lunik-Torrent
    $ npm install
    $ nano src/config.json
##### Run
    $ npm start

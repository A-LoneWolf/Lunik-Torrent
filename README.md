## ===== Installation =====
## Git

```
$ git clone https://github.com/Lunik/Lunik-Torrent.git
$ cd Lunik-Torrent
$ npm install
```

Configuration into: `configs/config.json`


### Run

```
$ npm start
or with forever
$ npm run forever-start
```
